ALL COMMANDS

📁 Repository Setup:
git init — Initialize a new Git repository
git remote add origin <repository_url> — Link local repo to GitHub
git clone <repository_url> — Clone an existing repository

📝 Staging & Committing Changes:
git add . — Add all changes to staging
git add <filename> — Add specific file to staging
git status — Check current status of changes
git commit -m "Your message" — Commit staged changes
git commit -am "Your message" — Add & commit tracked files in one step

🚀 Pushing Changes:
git push -u origin main — First push to the main branch
git push — Push changes to the linked branch
git push origin <branch_name> — Push to a specific branch
git push -f — Force push (use with caution)

🌿 Branching & Collaboration:
git branch — List all branches
git checkout -b <new_branch> — Create and switch to a new branch
git checkout <branch_name> — Switch to an existing branch
git pull — Pull latest changes from GitHub
git pull origin <branch_name> — Pull specific branch
git merge <branch_name> — Merge another branch into current one

🗑️ Fixes & Cleanups:
git rm <filename> — Remove file from repo
git commit --amend — Edit last commit
git reset --hard HEAD~1 — Undo last commit
git stash — Save uncommitted changes temporarily
git stash pop — Apply stashed changes
